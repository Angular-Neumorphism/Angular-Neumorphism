About

Neomorphism is a quite a new UI trend. It's kind of 'soft design' that mostly uses shadows to do things. Using a Neomorphism in real projects is quite challenging because of not very clear rules and accessibility issues. But despite this, Neomorphism is beautiful. So we've decided to create this open-source framework to do using Neomorphism easier for developers.

Now we're on the very first stage - creating design concepts for the components. Further, step by step, we're going to create independent reusable modules for each of them. Now, if you like some component fill free to use it styles from our GitHub repository.

With huge love and respect, we've decided to use Angular Material as a base for our framework. So if you'll use any component be sure to install all required dependencies.

If you have any trouble, questions, suggestions, or want to contribute Angular Neomorphism framework, don't hesitate to contact us.
