Neomorph checkbox 
    import { NeoCheckboxModule } from '@neomorphism/ng-neomorphism/neo-checkbox'
<neo-checkbox> provides the same functionality as a HTML5 native checkbox except indeterminate behavior. It could work with template-driven and reactive forms. Have the same API as Material checkbox and additional mixins for styling.
Basic checkbox 
        <neo-checkbox checked="true">Option checked</neo-checkbox>
        <neo-checkbox >Option two</neo-checkbox>
        <neo-checkbox disabled>Option disabled</neo-checkbox>

Styling

Because Neomorphism uses additional shadows, changing background color will cause mismatching between the background color and basic shadows. So we provide additional API to solve this issue. Use neo-checkbox-background($color) mixin to establish shadow colors automatically according to background. Just pass your current background color to this mixin.
Checkbox with custom background, shadows and check mark 
      
      <neo-checkbox 
        checkMarkColor="#f3f3f3" 
        class="checkbox red" 
        checked="true">Option checked</neo-checkbox>
      
      <neo-checkbox 
        checkMarkColor="#f3f3f3" 
        class="checkbox green" >Option two</neo-checkbox>
      
      <neo-checkbox 
        checkMarkColor="#f3f3f3" 
        class="checkbox red" 
        disabled>Option disabled</neo-checkbox>
        
        @import "~@neomorphism/ng-neomorphism/mixins/neo-checkbox.mixins"; 
        
          .checkbox { 
              &.green {
              @include neo-checkbox-background(#5fda92);
              }
          }
