Basic Input Controls that collect and validate user input.
Input with validation Email is not valid Disabled input Textarea 
    import {Component, AfterViewInit} from '@angular/core'
    import {FormControl, FormGroup, Validators} from '@angular/forms'

    @Component({
      selector: 'app-inputs',
      templateUrl: './inputs.component.html', 
      styleUrls: ['./inputs.component.scss'],  
      }) 
    export class InputsComponent implements AfterViewInit  {
    public textInput = '';

      exampleForm = new FormGroup({
      requiredInput: new FormControl('', [Validators.email]),
      disabledInput: new FormControl({value: 'Value from formcontrol', 
                                    disabled: true}),
      textInput: new FormControl(''),
      });
      constructor() {}
      }
     
        
    <form [formGroup]="exampleForm"> 
        <neo-form-field class="form-field">
          <neo-label> Input with validation</neo-label> 
          <input
            neoInput
            formControlName="requiredInput"
            autocomplete="off"
            placeholder="name@company.com" /> 
          <neo-error *ngIf="exampleForm.invalid"> Email is not valid</neo-error>
        </neo-form-field>

        <neo-form-field class="form-field">
          <neo-label> Disabled input</neo-label> 
          <input
            neoInput
            formControlName="disabledInput"
            autocomplete="off"
            disabled
            placeholder="It's a placeholder" /> 
        </neo-form-field>

        <neo-form-field class="form-field">
          <neo-label> Textarea</neo-label> 
          <textarea
            neoInput
            formControlName="disabledInput"
            autocomplete="off"
            placeholder="It's a placeholder" ></textarea>  
        </neo-form-field>
    </form>

Additional to Material API we provide some scss mixins to manage form-field and input background.

@include neo-form-field-background($color)

Using at form-field element. It set backgroung-color of form-field and set input color to 5% darker Also it set input shadows denending on provided background color
Styled formfield Input styled with neo-form-field-background($color) mixin 
        <neo-form-field class="form-field styled-formfield">
          <input
            neoInput
            autocomplete="off"
            placeholder="It's a placeholder" /> 
        </neo-form-field>
        
        @import "../../components/neo-form-field/neo-form-field/form-field.theme"; 
        
          .styled-formfield { 
            @include neo-form-field-background(#f84f5a);
            border-radius: 8px;
            display: flex;
            align-items: center;
            flex: 1;
            height: 200px;
             }

Uses at input element. It set backgroung-color of input

@include neo-input-background($color);

When using with neo-form-field-background($color) it's override input background-color
