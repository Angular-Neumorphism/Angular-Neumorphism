Neomorph tabs 
    import { NeoFormTabsModule }  from '@neomorphism/ng-neomorphism/neo-tabs'
<neo-tab-grout> Neomorph styled tabs organize content into separate views where only one view can be visible at a time. Each tab's label is shown in the tab header and the active tab's label is designated with the animated ink bar. When the list of tab labels exceeds the width of the header, pagination controls appear to let the user scroll left and right across the labels.
