Neomorph ripple 
      import { NeoRippleModule }from '@neomorphism/ng-neomorphism/neo-ripple'
neoRipple connect user input to screen reactions by using ripples to both indicate the point of touch, and to confirm that touch input was received. For touch or mouse, this occurs at the point of contact. Also you could use matRipple without additional modules import. To provide one type ripple for whole application import RIPPLE_TYPE and RIPPLE_TYPES from NeoRippleModule and use { provide: RIPPLE_TYPE, useValue: RIPPLE_TYPES.NEO } or { provide: RIPPLE_TYPE, useValue: RIPPLE_TYPES.MATERIAL } at app module providers section
