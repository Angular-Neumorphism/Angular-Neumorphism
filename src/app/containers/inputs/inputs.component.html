    import {Component, AfterViewInit} from '@angular/core'

    @Component({
      selector: 'app-inputs',
      templateUrl: './inputs.component.html', 
      styleUrls: ['./inputs.component.scss'],  
      })

      exampleForm = new FormGroup({
      requiredInput: new FormControl('', [Validators.email]),
      disabledInput: new FormControl({value: 'Value from formcontrol',

      }
     
        
    <form [formGroup]="exampleForm">

Styling

Additional to Material API we provide some scss mixins to manage form-field and input background. Neomorphism uses additional shadows, so change of a background color will cause mismatching between the background color and basic shadows. So we provide additional API to solve this issue.

@include neo-form-field-background($color)

Use it at form-field element. It set backgroung-color of form-field and set input color to 5% darker. Also it set input shadows denending on provided background color.

@include neo-input-background($color);

Could be used to establish input background and shadows. When using with neo-form-field-background($color) it's override input background-color.
