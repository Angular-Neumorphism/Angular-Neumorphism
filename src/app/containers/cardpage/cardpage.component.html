Neomorph card 
    import { NeoCardModule }  from '@neomorphism/ng-neomorphism/neo-card';
<neo-card>is a content container.While this component can be used alone, it also provides a number of preset styles for common card sections, including:
  • <neo-card-header>
  • <neo-card-title>
  • <neo-card-subtitle>
  • neo-card-avatar (directive)>
  • <neo-card-content>
  • <neo-card-actions>
  • neo-card-image (directive)>
  • <neo-card-footer>
